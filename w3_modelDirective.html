The ng-model directive binds the value of HTML controls (input. selct, textarea) to application data

The ng-model Directive

With the ng-model directive you can bind teh value of an input field to a variable created in AngularJS

Two-way Binding

The binding goes both ways. If the user changed the value inside the input field, the AngularJS property will also change its value

Validate User Input

The ng-model directive can procide type validation for application data (number, e-mail, required)

Application Status

The ng-model directive can provide status for appliction data (invalid, dirty, touched, error)

CSS Classes

The ng-model directive provides CSS classes for HTML elements, depending on their status:

The ng-model directive adds/removes the following classes, accourding to the status of the form field
  • ng-empty
  • ng-not-empty
  • ng-touched
  • ng-untouched
  • ng-valid
  • ng-invalid
  • ng-dirty
  • ng-pending
  • ng-pristine
