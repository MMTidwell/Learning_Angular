AngularJS Scope

The scope is the binding part between the HTML (view) and the JS (controller).
The scope is an obj with the available properties and methods.
The scope is available for both the view and the controller.

How to use the Scope

Understanding the Scope

If we consider an ANgularJS application to consist of:
  • View, which is the HTML
  • Model, which is the dat available for the current view
  • Controller, which is the JS function that makes/changes/removes/contrls the data
The Scope is the model

Know your Scope

It is important to know which scope you are dealing with, at any time.

Root Scope

All applications have a $rootScope which is the scope created on the HTML elemnet that contains the ng-app directive.
The rootScope is available in the entire application
If a var has the same name in both the current scope and in the rootScope, the application uses the one in the current scope.
